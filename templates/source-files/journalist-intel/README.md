# Journalist Intelligence Inputs

Use this folder for selected-angle journalist collection.

This research happens after:

1. `05-beats.md` is completed.
2. The user chooses one or more outreach angles and one active selected angle package is clear for the next run.
3. `Selection status` is set to exact `confirmed`.

Muck Rack, SERP, outlet-page, contact-source, and manual profile research belong here before `06-journalist-intel.md` is written.

- `selected-angle/`: confirmed angle summary, query plan, SERP notes, outlet-page notes, contact-review notes, and collection log
- `muck-rack-exports/`: selected-angle Muck Rack search results, profile captures, and article captures
- `profile-notes/`: manually reviewed journalist notes and profile review notes
- `bulk-beat-collection/`: selected-angle bulk Muck Rack collection artifacts when the confirmed lane requires scale
- `rejected-or-stale-sources.md`: old, cross-campaign, wrong-beat, wrong-geography, or otherwise unusable sources

Do not store all-angle collection here unless the user explicitly overrides the selected-angle workflow.
