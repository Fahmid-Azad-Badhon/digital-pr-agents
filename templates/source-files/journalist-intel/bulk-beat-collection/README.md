# Bulk Beat Collection

Use this folder only when the confirmed selected angle uses `Bulk Muck Rack lane` or `Hybrid lane`.

This folder is for selected-angle bulk discovery, not all-angle collection.

Expected generated artifacts may include:
- selected-angle bulk query JSON
- per-beat JSON exports
- per-beat Markdown summaries
- combined CSV
- deduped CSV
- `summary-all.json`
- `summary-all.md`

After any resume or partial rerun, rebuild combined summaries from the full selected-angle JSON set before treating the run as final.
