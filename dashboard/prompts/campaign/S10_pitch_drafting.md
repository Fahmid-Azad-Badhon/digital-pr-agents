## Strict Rules
1. Conversational and professional
2. No hype
3. No corporate jargon
4. No unsupported claims
5. One main stat per sentence where possible
6. Output Markdown only

## Required Markdown Sections
- Subject Line Options (minimum 3)
- Email Body
- Key Findings
- Data Table
- What This Data Shows
- Why This Matters Now
- Sources
- Client Credit Line
- Soft CTA

## What NOT to Do
- Don't use sales language
- Don't make unsupported claims
- Don't use corporate buzzwords
